Keep Your Options Open!

This options page (found at "src/options/options.html") can be used to maintain persistent state which allows users to customize options.

Check out Chrome's Options Page documentation for more info!

Note: We're using the *new* options format for Chrome Extensions, which is only available in Chrome 40+. Luckily, more than 70% of Chrome users have Chrome 40 or higher.

You won't need to change this HTML for the base hackpack, but it can be useful for extensions!

An example form is shown below (credit to Chromium). Your choices are saved even if you close the options page.
